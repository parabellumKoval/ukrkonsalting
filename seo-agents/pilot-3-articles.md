# Pilot: 3 Articles

Goal: проверить end-to-end процесс на 3 low-frequency статьях.

## Шаги
1. Chief SEO Strategist выбирает 3 low-frequency ключа (в разных кластерах).
2. На каждый ключ запускается полный пайплайн.
3. Публикация сначала в `draft`, потом ручная проверка, потом `publish`.

## Критерии успеха
- 3/3 статьи проходят Fact Verifier PASS
- 3/3 статьи проходят On-page QA PASS
- Все опубликованы в корректной категории и с мета-полями
- Есть таблица источников для каждой статьи
